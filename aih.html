AIH:

É a Autorização de Internações Hospitalares, plataforma na qual o município tem que informar as internações hospitalares, motivos e quantos dias o cidadão ficou sob tratamento.

Após informar no sistema, a ficha de internação contendo todos os dados é processada, e para cada motivo específico (CID), é repassado um valor diferente para custear o tratamento durante a permanência do usuário na rede de internação.

A empresa oferece-se para lançar os procedimentos em suas etapas, alimentando o sistema mensalmente com dados dos pacientes internados, para arrecadar mais recursos para o município.
